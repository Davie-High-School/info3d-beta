### Overview 
![ONSHAPE](public/cad/onshape.png)

Onshape is a browser based parametric Computer aided design software. 
The free acoount has no limmitation, however every model created is accessable to the public. 

On Shape functions well in a web browser and functions on All devices with internet access. 
- There is a dedicated iPad version with Apple Pincel Support
