
# Formatting Capabilities in Nextra

This document demonstrates the various **formatting capabilities** you can use with **MDX** in **Nextra**.

## 1. **Headings**

Headings are used to structure your content. You can create different levels of headings by using `#`.

### Level 1 Heading

```mdx
# This is a Level 1 Heading
```

#### Level 2 Heading

```mdx
## This is a Level 2 Heading
```

##### Level 3 Heading

```mdx
### This is a Level 3 Heading
```

##### Level 4 Heading

```mdx
#### This is a Level 4 Heading
```

##### Level 5 Heading

```mdx
##### This is a Level 5 Heading
```

##### Level 6 Heading

```mdx
###### This is a Level 6 Heading
```

## 2. **Text Formatting**

You can style your text using various Markdown syntax.

### Bold Text

```mdx
**This is bold text**
```

It looks like this: **This is bold text**

### Italics Text

```mdx
*This is italic text*
```

It looks like this: *This is italic text*

### Strikethrough Text

```mdx
~~This is strikethrough text~~
```

It looks like this: ~~This is strikethrough text~~

### Inline Code

```mdx
`console.log("Hello, World!")`
```

It looks like this: `console.log("Hello, World!")`

---

## 3. **Lists**

You can create both **ordered** and **unordered lists**.

### Unordered List

```mdx
- Item 1
- Item 2
  - Subitem 1
  - Subitem 2
- Item 3
```

It looks like this:

- Item 1
- Item 2
  - Subitem 1
  - Subitem 2
- Item 3

### Ordered List

```mdx
1. First item
2. Second item
3. Third item
```

It looks like this:

1. First item
2. Second item
3. Third item

---

## 4. **Links**

You can create links in MDX.

### External Link

```mdx
[Visit Google](https://www.google.com)
```

It looks like this: [Visit Google](https://www.google.com)

### Internal Link (to a page within your site)

```mdx
[Go to About Page](/about)
```

It looks like this: [Go to About Page](/about)

### Anchor Link (link to a specific section in the same page)

```mdx
[Go to Section](#section-heading)
```

---

## 5. **Images and GIFs**

Images and GIFs can be easily added using the `![alt text](URL)` syntax.

### Image

```mdx
![3D Printing Image](./3d-printing.jpg)
```

It looks like this: ![3D Printing Image](./3d-printing.jpg)

### GIF

```mdx
![3D Printer Animation](./3d-printing.gif)
```

It looks like this: ![3D Printer Animation](./3d-printing.gif)

---

## 6. **Callouts**

Callouts are styled boxes used to highlight important content, such as tips or warnings.

### Info Callout

```mdx
:::info
This is an informational box. Use it to provide general information.
:::
```

It looks like this:
:::info  
This is an informational box. Use it to provide general information.  
:::

### Tip Callout

```mdx
:::tip
This is a tip box. Use it to highlight useful tips or best practices.
:::
```

It looks like this:
:::tip  
This is a tip box. Use it to highlight useful tips or best practices.  
:::

### Warning Callout

```mdx
:::warning
This is a warning box. Use it to warn users about potential issues.
:::
```

It looks like this:
:::warning  
This is a warning box. Use it to warn users about potential issues.  
:::

---

## 7. **Tables**

Tables are useful for displaying data in a structured format.

### Example Table

```mdx
| Feature | Description         |
| ------- | ------------------- |
| 3D Print | Additive manufacturing |
| CNC      | Subtractive manufacturing |
```

It looks like this:

| Feature    | Description             |
| ---------- | ----------------------- |
| 3D Print   | Additive manufacturing  |
| CNC        | Subtractive manufacturing|

---

## 8. **Code Blocks**

You can add code blocks for displaying code snippets.

### Inline Code

```mdx
Use the `console.log` function to print to the console.
```

It looks like this: Use the `console.log` function to print to the console.

### Block Code

```mdx
```javascript
const message = "Hello, World!";
console.log(message);
```
```

It looks like this:

```javascript
const message = "Hello, World!";
console.log(message);
```

---

## 9. **Embedding YouTube Videos**

You can embed YouTube videos by using an iframe.

```mdx
<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allowfullscreen></iframe>
```

It will embed a YouTube video like this:

<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allowfullscreen></iframe>

### Conclusion

This document covers the key formatting capabilities in **MDX with Nextra**:
- **Headings, Lists, Links, Images, GIFs**
- **Callouts** for highlighting important information
- **Code blocks** for displaying code
- **Tables** for structured data
- **Embedded content** like YouTube videos
- **Custom components** (React components inside MDX)

You can use these formatting capabilities to create a well-structured and interactive documentation site with Nextra.
